# dotfiles
My configuration files for command line utilities.

Requires GNU stow.

Clone the repo and run `stow -t ~ .` to create the symlinks.

## Requirements
### Neovim
 - `clangd`
 - `node`
 - `npm`
 - `python`
 - `fd`
 - `lazygit`

### Sway
 - `walker`
 - `foot`
 - `dolphin`
 - `xdg-desktop-portal-hyprland`
 - `xdg-desktop-portal-gtk`
